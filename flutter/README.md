# Flutter

## Common flutter command
### To add package
```cmd
 flutter pub add <package>
```
### To remove package
```cmd
 flutter pub remove <package>
```
### To add package to dev
```cmd
 flutter pub add --dev <package>
```
## To test project
```cmd
flutter test <test project path>
```
